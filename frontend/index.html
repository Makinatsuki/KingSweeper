KingSweeper

How to Play

KingSweeper is a logic puzzle game where you need to clear a grid without hitting any hidden mines.

  • Left Click: Reveal a tile. If it's a mine, the game is over. If it's a number, it tells you how many mines are adjacent to that tile.
  • Right Click: Place a flag on a tile you suspect contains a mine.
  • Objective: Clear all tiles that do not contain mines.
